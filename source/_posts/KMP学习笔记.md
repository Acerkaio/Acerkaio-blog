### 重新定义水博客

border[y] = x 表示 b[1 ~ x] = b[(y - x + 1) ~ y]

```cpp
inline void find_border() {
    int j = 0;
    for (int i = 2; i <= lb; i++) {
        while (b[i] != b[j + 1] && j) { //当失配时
            j = border[j];
        }

        if (b[i] == b[j + 1]) { //可以加入一个 
            j++;
        }
        border[i] = j;  //储存 
    }
}
```
